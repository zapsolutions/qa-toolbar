Features

Create ToDos & Entries

Quickly create todos and entries from your development site.

View Current ToDos & Entries

Keep up with current tasks at the click of a button.

Restrict by Domain & IP

Give access to yourself, or add the whole team!

Settings

Configure settings that are stored in JavaScript cookies.

 

Technology Used

MySQL Database

Used to store settings for IP & domain restriction

CakePHP (MVC Framework)

CakePHP is the framework that Maestro is written in. This plugin required the creation of AJAX accessible actions to interact with the database.

jQuery

The plugin primarily uses jQuery to make coding with JavaScript easier. The plugin will load jQuery if it does not find it on the page.

HTML/CSS + Twitter Bootstrap + Angle Admin theme

The styling of the plugin uses the same theme as Maestro, to give it a familiar look & feel.
